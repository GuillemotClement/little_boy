## BAT 
Script pour Windows 
- `importDB`: viens supprimer et importer une base de donnée. Permet de récupérer une base exportée, et l'importer directment dans son MariaDB local. Le nom du fichier doit être celui du fichier .sql généré par l'exportation. Le nom de la nouvelle base local se base sur le nom du fichier fournis.
